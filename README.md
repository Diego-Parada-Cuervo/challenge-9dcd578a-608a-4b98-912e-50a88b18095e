# Gestión de estado global en aplicación web

En una aplicación web de finanzas personales, los usuarios gestionan sus gastos y ahorros. La aplicación necesita mantener un estado global que refleje los cambios en tiempo real en diferentes secciones. El objetivo es implementar un sistema de gestión de estado que permita a los usuarios ver y actualizar su información financiera de manera consistente.
